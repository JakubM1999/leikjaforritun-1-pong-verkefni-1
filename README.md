A self-paced, interactive browser-based guide designed to introduce students to game development using the **Godot Engine (v4)**.

This course takes students from "zero" to a fully functional **Pong** clone, covering the essentials of the engine, GDScript, and game logic.

## üìö Course Content

The material is divided into two main modules accessible via the main menu:

1.  **Introduction (`intro.html`):**
    *   Why use Godot?
    *   Installation (Standard Version).
    *   Interface overview (Scene Dock, FileSystem, Inspector).
    *   Core concepts: Nodes & Scenes.

2.  **Pong Guide (`pong-guide.html`):**
    *   **Step-by-Step Tutorial:** Building the game from scratch.
    *   **Setup:** Project creation and window settings.
    *   **Physics:** Creating the Paddle and Ball (CharacterBody2D).
    *   **Scripting:** Basic GDScript for movement and collision (`move_and_slide`, `move_and_collide`).
    *   **Logic:** Simple score tracking without complex signals.
    *   **Export:** How to publish the game to the web.

## üöÄ How to Use

### For Students
1.  Clone or download this repository.
2.  Open **`index.html`** in any modern web browser.
3.  Navigate through the slides using the **Arrow Keys** or the on-screen buttons.
4.  Follow the instructions in Godot alongside the slides.

### For Instructors
This repository serves as a template. The slides contain placeholder boxes for visuals (screenshots/GIFs).

**To finish setting up the course:**
1.  Run through the tutorial yourself in Godot.
2.  Take screenshots or screen recordings of each step.
3.  Replace the `<div class="placeholder-box">` elements in the HTML files (`intro.html` and `pong-guide.html`) with standard `<img>` or `<video>` tags.
