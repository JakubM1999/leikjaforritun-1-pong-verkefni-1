PONG: SKREF FYRIR SKREF

LEIÐBEININGAR

Hér munum við byggja leikinn frá grunni.


ÝTTU Á SPACE TIL AÐ BYRJA

SKREF 1: NÝTT VERKEFNI

Byrjum á að búa til hreint verkefni.


  1. Opnaðu Godot.
  2. Smelltu á New Project.
  3. Veldu staðsetningu (Browse) og búðu til möppu sem heitir "Pong".
  4. Veldu Mobile renderer (það er léttara fyrir 2D leiki).
  5. Smelltu á Create & Edit.
[PLACEHOLDER: SCREENSHOT OF PROJECT CREATION WINDOW]

SKREF 2: STILLA SKJÁINN

Pong er spilaður á ákveðnu vallarsvæði. Stillum upplausnina.


  1. Farðu í Project -> Project Settings í efstu valmyndinni.
  2. Í General flipanum, leitaðu að Display -> Window.
  3. Stilltu Viewport Width á 1152 og Viewport Height á 648 (eða þínar tölur).
  4. Skrunaðu niður í Stretch og stilltu Mode á canvas_items.
[PLACEHOLDER: SCREENSHOT OF PROJECT SETTINGS WINDOW]

SKREF 3: MAIN SCENE

Búum til aðal sviðið þar sem leikurinn fer fram.


  1. Í Scene Dock (vinstra megin), smelltu á 2D Scene hnappinn. Þetta býr til Node2D sem rót.
  2. Tvísmelltu á nafnið Node2D og endurnefndu það í Main.
  3. Ýttu á Ctrl + S (eða Cmd + S) til að vista.
  4. Vistaðu skrána sem main.tscn.
[PLACEHOLDER: SCREENSHOT OF SCENE DOCK WITH 'MAIN' NODE]

SKREF 4: BAKGRUNNUR

Gerum völlinn svartan.


  1. Smelltu á Main node-ið svo það sé valið.
  2. Smelltu á + takkann (Add Child Node) eða ýttu á Ctrl + A.
  3. Leitaðu að ColorRect og búðu það til.
  4. Í Inspector (hægra megin), veldu Color og settu það í svartan.
  5. Dragðu hornin á kassanum í Viewport svo hann þeki allan skjáinn (eða notaðu Anchor preset: Full Rect).
[PLACEHOLDER: SCREENSHOT OF COLORRECT IN VIEWPORT]

SKREF 5: SPAÐINN (PADDLE)

Nú þurfum við leikmann. Við gerum nýtt Scene fyrir hann.


  1. Smelltu á + flipann við hliðina á "Main" flipanum fyrir ofan Viewport til að opna nýtt tómt Scene.
  2. Veldu Other Node og leitaðu að CharacterBody2D (fyrir Godot 4) eða KinematicBody2D (Godot 3). Við notum Godot 4: CharacterBody2D.
  3. Nefndu það Paddle.
  4. Bættu við tveimur börnum (children): ColorRect (fyrir útlitið) og CollisionShape2D (fyrir eðlisfræðina).

SKREF 6: ÚTLIT SPAÐANS

Látum spaðann líta út eins og spaða.


  1. Veldu ColorRect í Paddle scenu.
  2. Í Inspector, stilltu stærðina (Size) á 20 x 120.
  3. Stilltu stöðuna (Position) á -10 x -60 (svo hann sé miðjaður út frá rótinni).
  4. Veldu CollisionShape2D. Það mun vara við því að vanti lögun (Shape).
  5. Í Inspector, við hliðina á Shape, veldu New RectangleShape2D.
  6. Stilltu stærðina á CollisionShape þannig að hún passi við ColorRect.
[PLACEHOLDER: SCREENSHOT OF PADDLE SETUP]

SKREF 7: INNSLÁTTUR (INPUT MAP)

Við þurfum að skilgreina hvaða takkar hreyfa spaðann. Við gerum það í Input Map.


  1. Farðu í Project -> Project Settings.
  2. Veldu Input Map flipann.
  3. Efst, skrifaðu move_up í "Add New Action" og smelltu á Add.
  4. Endurtaktu fyrir move_down.
  5. Smelltu á + takkann við hliðina á nýju aðgerðunum til að bæta við takka.
  6. Fyrir move_up, veldu "Physical Key" og ýttu á W takkann.
  7. Fyrir move_down, veldu "Physical Key" og ýttu á S takkann.

Við munum nota þessar aðgerðir í kóðanum til að hreyfa leikmann 1. Hægt er að bæta við fleiri tökkum (eins og örvunum) fyrir leikmann 2 síðar.

[PLACEHOLDER: SCREENSHOT OF INPUT MAP SETTINGS]

SKREF 8: KÓÐI FYRIR SPAÐANN

Nú er kominn tími til að láta spaðann hreyfast.


  1. Hægrismelltu á Paddle node-ið í Scene Dock og veldu Attach Script.
  2. Gakktu úr skugga um að Template sé "Default".
  3. Smelltu á Create.

Skiptu út öllum kóðanum í skriftunni með eftirfarandi:

extends CharacterBody2D

@export var speed: int = 300

func _physics_process(delta):
    # Fá stefnu frá Input Map
    var direction = Input.get_axis("move_up", "move_down")

    # Stilla hraða (velocity)
    velocity.y = direction * speed

    # Hreyfa leikmanninn
    move_and_slide()

Farðu í "Main" scene, settu (dragðu) "paddle.tscn" inn á sviðið og keyrðu leikinn (F5). Þú ættir nú að geta hreyft spaðann með W og S!

SKREF 9: BOLTINN (THE BALL)

Enginn Pong án bolta. Búum til nýtt Scene.


  1. Búðu til nýtt Scene. Rótin (Root Node) skal vera CharacterBody2D.
  2. Nefndu það Ball.
  3. Bættu við ColorRect (stærð 10x10).
  4. Bættu við CollisionShape2D (RectangleShape2D, 10x10).
  5. Vistaðu sem ball.tscn.

SKREF 10: KÓÐI BOLTANS

Boltinn þarf að skoppa. Við notum move_and_collide.


Settu skriftu á Ball og notaðu þennan kóða:

extends CharacterBody2D

var speed = 400
var direction = Vector2.ZERO

func _ready():
    # Velja handahófskennda átt í byrjun
    direction.x = [1, -1].pick_random()
    direction.y = [1, -1].pick_random()
    velocity = direction * speed

func _physics_process(delta):
    # move_and_collide skilar upplýsingum um árekstur
    var collision = move_and_collide(direction * speed * delta)
    
    if collision:
        # Ef árekstur verður, látum boltann skoppa (bounce)
        direction = direction.bounce(collision.get_normal())

SKREF 11: VEGGIR (WALLS)

Við viljum ekki að boltinn fari út af skjánum uppi eða niðri.


  1. Opnaðu Main scene.
  2. Bættu við StaticBody2D node og nefndu það Walls.
  3. Bættu við tveimur CollisionShape2D börnum við Walls.
  4. Fyrir þann fyrri: Notaðu WorldBoundaryShape2D. Snúðu honum þannig að línan snúi NIÐUR (Normal 0, 1) og staðsettu efst á skjánum.
  5. Fyrir þann seinni: Notaðu WorldBoundaryShape2D. Snúðu honum þannig að línan snúi UPP (Normal 0, -1) og staðsettu neðst á skjánum.

SKREF 12: ANDSTÆÐINGUR (AI)

Við þurfum andstæðing. Við getum endurnýtt spaðann en breytt kóðanum.


  1. Í FileSystem, hægrismelltu á paddle.tscn og veldu Duplicate. Nefndu það enemy.tscn.
  2. Opnaðu enemy.tscn.
  3. Hægrismelltu á rótina (Node) og veldu Detach Script (til að fjarlægja stýringuna).
  4. Bættu við nýrri skriftu:
extends CharacterBody2D

var speed = 250
var ball

func _ready():
    # Finnum boltann í leiknum
    ball = get_parent().get_node("Ball")

func _physics_process(delta):
    if ball:
        # Færa spaðann í átt að Y-staðsetningu boltans
        var direction = (ball.position - position).normalized()
        velocity.y = direction.y * speed
        move_and_slide()

SKREF 13: LOKA SAMSETNING

Setjum allt saman í Main Scene.


  1. Opnaðu Main scene.
  2. Dragðu Ball.tscn inn á miðjan skjáinn.
  3. Dragðu Paddle.tscn vinstra megin.
  4. Dragðu Enemy.tscn hægra megin.
  5. Ýttu á F5 til að spila!

Boltinn ætti nú að skoppa af veggjum og báðum spöðum.

SKREF 14: STIGAGJÖF (LOGIC)

Við þurfum ekki flókna "nodes" til að telja stig. Við getum einfaldlega skoðað staðsetningu boltans í kóða.


Skjárinn okkar er 1152 pixlar á breidd.

  • Ef Ball.position.x > 1152: Boltinn fór út hægra megin -> Vinstri (Player) skorar.

SKREF 15: VIÐMÓT (UI)

Sýnum stigin.


  1. Í Main scene, bættu við CanvasLayer node (þetta lætur textann fljóta ofan á öllu).
  2. Bættu við Label node undir CanvasLayer.
  3. Skrifaðu "0 - 0" í Text reitinn í Inspector.
  4. Í Label Settings, stækkaðu letrið (Font Size) í t.d. 64px og settu það efst fyrir miðju.

SKREF 16: LEIKREGLUR (MAIN SCRIPT)

Tengjum allt saman í kóða.


  1. Settu skriftu á Main node-ið.
  2. Notaðu þennan kóða (við skoðum hvar boltinn er í hverjum ramma):
extends Node2D

var score_player = 0
var score_enemy = 0

func _process(delta):
    # Ef boltinn fer út vinstra megin
    if $Ball.position.x < 0:
        score_enemy += 1
        update_ui()
        reset_ball()

    # Ef boltinn fer út hægra megin (miðað við skjábreidd 1152)
    if $Ball.position.x > 1152:
        score_player += 1
        update_ui()
        reset_ball()

func update_ui():
    $CanvasLayer/Label.text = str(score_player) + " - " + str(score_enemy)

func reset_ball():
    $Ball.position = Vector2(1152/2, 648/2) # Miðja skjás
    # Endurstilla áttina (valfrjálst, en gott til að hann fari ekki strax aftur í mark)
    $Ball.direction.x = [1, -1].pick_random()
    $Ball.direction.y = [1, -1].pick_random()

SKREF 17: SKIL Á VERKEFNI

Hvernig á að skila verkefninu inn á Innu.


  1. Lokaðu Godot (mjög mikilvægt svo allar skrár vistist rétt).
  2. Finndu möppuna þar sem verkefnið þitt er (t.d. Documents/Godot/Pong).
  3. Hægrismelltu á möppuna og veldu Compress eða Send to -> Compressed (zipped) folder.
  4. Skilaðu .zip skránni inn á Innu.

Athugið: Kennari þarf að sjá kóðann þinn til að gefa einkunn, þess vegna skillum við verkefnamöppunni en ekki útfluttum leik.

BONUS: DEILA LEIKNUM (EXPORT)

Viltu sýna vinum þínum leikinn? Hér er hvernig þú býrð til vefútgáfu.


  1. Farðu í Project -> Export.
  2. Smelltu á Add... og veldu Web (fyrir vafra) eða Windows/Mac.
  3. Ef þú færð villumeldingu um "Export Templates", smelltu á Manage Export Templates og sæktu þau.
  4. Smelltu á Export Project.
  5. Vistaðu index.html (fyrir vef) í nýja möppu.
